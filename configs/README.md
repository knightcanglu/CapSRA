# Config Presets

This directory provides ready-to-edit configuration templates for common CapSRA experiment settings.

## Included Presets

- `clip_mainline.yaml`
- `vilt_mainline.yaml`
- `prompthate_patch.yaml`
- `procap_patch.yaml`

These files are documentation-oriented presets. Copy the command arguments you need into your own shell scripts, launchers, or experiment manager.
